## 🚀 Project Overview

End-to-end **data engineering pipeline** that transforms raw e-commerce data into a structured **PostgreSQL data warehouse (star schema)** using Python.

This project demonstrates:

- ETL pipeline design
- Data cleaning & validation
- Star schema modeling
- Large-scale data loading (800k+ rows)
- Data quality checks
